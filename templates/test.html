MPIN Test Cases

Below are different types of PIN test cases to help evaluate the analyzer. These test inputs cover various patterns such as sequential numbers, common PINs, and demographic-based PINs.

Valid PIN Examples (Strong)

PIN: 7392

Type: 4-digit

Reason: Randomized, not demographic or sequential

PIN: 824193

Type: 6-digit

Reason: Uncommon digits, no personal tie-ins

Weak PIN Examples (Common)

PIN: 1234

Type: 4-digit

Reason: Sequential and commonly used

PIN: 111111

Type: 6-digit

Reason: Repeated digits, easily guessable

Demographic-Based PINs

PIN: 2301

DOB: 2001-01-23

Reason: Reversed day-month from DOB

PIN: 121212

Anniversary: 2012-12-12

Reason: Direct match to important date

Invalid Inputs

PIN: 12a4

Error: Contains non-numeric character

PIN: 12345

Error: Invalid length (not 4 or 6)

These test cases demonstrate how the MPIN Analyzer classifies PINs based on patterns and personal information. For stronger security, always avoid predictable numbers.
